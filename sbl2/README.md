sbl2
========================================

SBL2是被SBL1从eMMC上加载到MIMEM/GMEM(0x2E000000 ~ 0x30000000)中执行的.

架构
----------------------------------------

* Krait

加载
----------------------------------------

* 载体: eMMC

执行
----------------------------------------

* 载体: MIMEM/GMEM (0x2E000000 ~ 0x30000000, 32MB)
* 起始地址: 0x2E000000

功能
----------------------------------------

* 初始化Krait处理器;
* 使能data cache 和 MMU;
* 加载和认证TZ image和RPM firmware;
* 配置外部DDR存储器;
* 加载和认证SBL3;
