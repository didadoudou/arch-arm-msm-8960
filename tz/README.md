TZ
========================================

TZ是被SBL2从eMMC加载到System IMEM(0x2A000000 ~ 0x2C000000)中的.

架构
----------------------------------------

* Krait

加载
----------------------------------------

* 载体: eMMC

执行
----------------------------------------

* 载体: System IMEM (0x2A000000 ~ 0x2C000000, 32MB)
* 起始地址: 0x2A000000

功能
----------------------------------------

* 安装安全环境(xPU)
* 提供安全模式支持ROT
