sbl1
========================================

SBL1是被RPM PBL从eMMC上加载到SYSTEM IMEM(0x2A000000 ~ 0x2C000000)
中执行的代码.

架构
----------------------------------------

* ARM7

加载
----------------------------------------

* 载体: eMMC

执行
----------------------------------------

* 载体: System IMEM (0x2A000000 ~ 0x2C000000, 32MB)
* 起始地址: 0x2A000000

功能
----------------------------------------

* 初始化SSBI和PMIC总线;
* 加载并认证SBL2;
* 配置Krait时钟并且将Krait复位.

### 过程

https://github.com/novelinux/arch-arm-msm-8960/tree/master/res/sbl1.png

完成上述功能之后, 复位后跳转到SBL2中执行.
